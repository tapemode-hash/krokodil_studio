🔘 Кнопки
Все кнопки используют класс .btn как базу. Offset-shadow (3–4px) — характерная деталь стиля. Hover поднимает кнопку на -2px.
Размеры
Варианты
CSS-правило
.btn-primary {
  background: var(--coral);
  color: white;
  box-shadow: 3px 4px 0 var(--coral-l);   /* характерная тень со смещением */
}
.btn-primary:hover {
  transform: translate(-1px, -2px);        /* "поднятие" при наведении */
  box-shadow: 4px 6px 0 var(--coral-l);
}
🃏 Карточки
Базовая карточка — .card. Stat-карточки имеют цветную рамку и offset-тень в тон. Декоративный круг в правом верхнем углу — opacity: .15.
